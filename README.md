# Ecommerce Sales Dashboard (Power BI)

## Overview
This Power BI dashboard analyzes ecommerce sales performance using a sample dataset.

## KPIs
- Total Orders
- Total Sales
- Total Quantity
- Total Profit

## Visualizations
- Month-wise Sales Trend
- Category-wise Sales
- Sales by Payment Mode
- State-wise Sales Map
- Delivery Status Analysis

## Tools Used
- Power BI
- Excel
- DAX
